NOTE ON THIS LIST
-----------------
This is a working reference document maintained by Administration and updated whenever extensions or on-call structures change. Report any incorrect number to Administration at ext 3050 so it can be corrected hospital-wide.


FREQUENTLY ASKED QUESTIONS
--------------------------
  * Q: This list has a wrong number - what do I do? A: Email Administration at admin-updates@sunrisemedical.internal or call ext 3050; do not just cross it out on the shared copy without reporting it.
  * Q: Is there a version of this list for personal cell phones? A: A abbreviated wallet card version is available from your unit manager for staff who want one; it excludes any number not appropriate for external distribution.
  * Q: Who updates the on-call schedule referenced in this document? A: The on-call scheduling app is maintained by each department's scheduling coordinator, not by Administration directly.

DISTRIBUTION NOTE
-----------------
This list is posted at every nurse station, every department front desk, and the House Supervisor office. It is reprinted whenever a number changes and quarterly regardless, to catch any accumulated small errors.


WORKED EXAMPLE - USING THIS LIST DURING AN AFTER-HOURS ISSUE
------------------------------------------------------------
11:40 PM - A nurse on the medical floor discovers the automated dispensing cabinet is offline and cannot dispense a scheduled medication.

11:41 PM - Nurse checks this list, finds IT Helpdesk is 24/7 at ext 6000, and calls to report the outage as urgent since it is actively affecting patient care.

11:45 PM - IT confirms a network issue affecting only that unit's ADC and provides an estimated fix time; nurse also notifies the House Supervisor at ext 3000 per unit policy for any medication access disruption.

11:50 PM - House Supervisor arranges a backup process (manual pharmacy delivery with two-nurse verification) until the ADC is restored, coordinating directly with the Pharmacy after-hours pager listed above.

This example shows why it's useful to know both the specific department extension AND the House Supervisor number - some issues need both a technical fix and an operational workaround while waiting.


HOW OFTEN THIS LIST IS VERIFIED
-------------------------------
Administration verifies every extension on this list at the start of each quarter by calling each line and confirming it reaches the intended department. Any number that rings incorrectly, goes to a disconnected line, or reaches the wrong department is corrected within one business day and an updated version is reprinted and redistributed to all posting locations.

Departments that change their main line number are required to notify Administration at ext 3050 as part of the IT change/move request process, so this list stays synchronized with actual phone system changes rather than relying solely on the quarterly audit to catch it.
